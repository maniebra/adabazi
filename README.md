# Adabazi

A single-page application for you to host you
local charades (i.e. adabazi in persian) games.

## Run the project

### Development

```sh
pnpm i
pnpm dev
```

### Build and Run

```shell
pnpm i
pnpm build
pnpm start
```

### Use Docker

```shell
docker compose up
```

## Sample inputs

All the inputs for this application should
be stored in files with `.txt` extension.

### Sample participants file

```text
John Doe-1
Ali Hoop-1
Rhino-1
Jimmy-1
```
